# RTScript – Extensión para GDEBA (No oficial)

![Logo del proyecto](assets/logo.png)

## 📌 Descripción
RTScript GDEBA es un **user script para Tampermonkey** que mejora la experiencia en la plataforma GDEBA.  
Incluye estilos personalizados, una botonera editable, modal para insertar contenido externo y un block de notas persistente.  
Su objetivo es **centralizar y simplificar la navegación interna y externa** mediante botones personalizables.

⚠️ **Aviso importante**: Esta herramienta es **no oficial** y fue desarrollada de manera independiente.  
No tiene relación con la Provincia de Buenos Aires ni con GDEBA.

---

## 🚀 Instalación
1. Instalar la extensión [Tampermonkey](https://www.tampermonkey.net/) en tu navegador.  
2. Descargar el script desde este repositorio.  
3. Importar el archivo en Tampermonkey.  
4. Recargar la plataforma GDEBA para ver los cambios.

---

## ✨ Funcionalidades
- **Botonera editable**: agregar, editar, borrar y reordenar enlaces.  
- **Modal interno**: abrir contenido web sin salir de GDEBA.  
- **Block de notas persistente**: guardar anotaciones personales.  
- **Backup/Restore**: exportar e importar tu configuración en JSON.  
- **Estilos visuales**: encabezados, botones y footer con diseño unificado.
